# GrantsProject — Анализ распределения грантов

Проект представляет собой Java-приложение для автоматизированной обработки данных о грантах, их структурированного хранения и визуализации аналитических показателей.

## Функционал
*   **Импорт данных:** Загрузка и парсинг информации из внешних источников.
*   **База данных:** Автоматическое создание локальной БД SQLite и управление связями между таблицами.
*   **Аналитика:** Расчет средних показателей рабочих мест по годам.
*   **Графики:** Визуализация данных с помощью библиотеки JFreeChart (сохранение в PNG и вывод на экран).

## Технологический стек
*   **Java 17** (или выше)
*   **Maven** (управление зависимостями)
*   **SQLite JDBC** (работа с БД)
*   **JFreeChart** (построение графиков)

## ⚙️ Запуск и установка

1.  **Клонируйте репозиторий:**
    ```bash
    git clone https://github.com/URFUMAN/GrantsProject
    ```
2.  **Соберите проект через Maven:**
    ```bash
    mvn clean install
    ```
3.  **Запустите приложение:**
    Файл базы данных `grants.db` будет автоматически создан в корневой папке проекта при первом обращении к БД.
